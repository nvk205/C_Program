| Lệnh                      | Dùng để làm gì                            |
| ------------------------- | ----------------------------------------- |
| `git init`                | Tạo Git repo trong folder                 |
| `git status`              | Xem trạng thái file                       |
| `git add .`               | Thêm tất cả file vào vùng chuẩn bị commit |
| `git commit -m "message"` | Lưu một phiên bản code                    |
| `git push`                | Đẩy code lên GitHub                       |
| `git pull`                | Lấy code mới từ GitHub về                 |
| `git clone <link>`        | Tải repo về máy                           |
| `git log --oneline`       | Xem lịch sử commit ngắn gọn               |
| `git remote -v`           | Xem repo GitHub đang kết nối              |
| `git branch`              | Xem nhánh hiện tại                        |
| `git branch -M main`      | Đổi tên nhánh chính thành main            |
